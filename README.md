# SDR\_Gen2

Prototype SDR for GOLF and FOX plus

Features:

**2 Channel Receiver**

&nbsp;	Common Input

&nbsp;	Alternative Separate Inputs

3 Selectable Input Filters

&nbsp;	VHF (2 mtrs to 500 MHz)

&nbsp;	S Band

&nbsp;	C Band

Std Mini-Circuits  Filters

&nbsp;	Substitutions available



**2 Channel Transmitter**

&nbsp;	Common Output

&nbsp;	Alternative Separate Outputs

2 Selectable Output Filters

&nbsp;	VHF (2 mtrs to 1296 MHz)

&nbsp;	S/C Band

Std Mini-Circuits  Filters

&nbsp;	Substitutions available

Output Power(s)

&nbsp;	½ to 1 watt (2 mtrs to 2700 MHz)

&nbsp;	~18 dBm (2 mtrs to 6000 MHz)

Experimental X Band Output

&nbsp;  ~ 5 dBm power output
